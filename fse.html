Python Raspberry Pi Sensors Gemini

In my Introduction to Engineering class we were challenged to design and build a device to help the visually impared be warned about objects that are above the waist. The reason this is important is because canes can only detect what is on the ground not above so a big safety concern is hitting their head.

Our design consisted of a camera and an ultrasonic sensor for detection and a vibration motor and buzzer for warnings. The web cam took a picture and sent the image to an OpenAI API asking if there were any objects in the vicinity. If the AI returned true the vibration motor would activate. The ultrasonic sensor was also working and if an object came within 1 foot the buzzer would go off, alerting the person to the danger.
