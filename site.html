Portfolio Site

HTML CSS JavaScript GitHub Pages

Designed and developed a custom portfolio website using HTML, CSS, and JavaScript to showcase engineering projects, technical skills, and experience. The site was built from scratch with a focus on clarity, usability, and maintainability.

Implemented a bold typographic layout on the homepage to immediately engage visitors, while maintaining a restrained color palette to communicate professionalism and technical competence. Design decisions were guided by readability and effective information hierarchy.

Deployed the site using GitHub Pages as a cost-effective and reliable hosting solution for static web applications.
